QlikSense: Calendar From-To Variables

We develop a QlikSense object Calendar done with jquery library. there are two variables for the dates From and To, that are changed the value everytime you select a date.

Qlik timeline

How to use:For QlikSense Desktop: put the project into the directory Documents/Qlik/Sense/Extensions

  • For QlikSense Server (QMC): import the project in to the extension.
  • Set two variables, the first one to be used as 'From variable' i.e initial date, and the second variable to be used as 'To Variable' i.e the final date, these variables can be used for calculating time range in your QlikSense app
  • N.B: the 'To' should be greater than or equal to 'From' variable.
